# Code Bloom Portfolio

A high-end, responsive portfolio website where precision engineering meets organic design. Built with a focus on minimalist aesthetics, smooth interactions, and a signature "pink & dark" color palette.

Live link: https://codebloom9.github.io/Portfolio/

## Features

- **Asymmetrical Project Grid**: A dynamic layout featuring 5 custom-curated projects with a "Featured Project" spotlight.
- **Glassmorphism UI**: Uses Tailwind's backdrop-blur and semi-transparent layers for a premium, modern feel.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive Marquee**: A custom "Featured Projects" scrolling strip decorated with animated spinning flowers.
- **Creative Process Section**: A 6-step roadmap detailing the journey from discovery to continuous support.
- **Stylish Pink Footer**: A deep-pink gradient footer integrated with blooming tree imagery and quick navigation.

## Tech Stack

* **HTML5**: Semantic structure.
-   **Tailwind CSS**: Utility-first styling for rapid, responsive UI development.
-   **JavaScript**: For smooth scrolling and interactive elements.
-   **Figma**: Used for the initial wireframing and design mockups.
